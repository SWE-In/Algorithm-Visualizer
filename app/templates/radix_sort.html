Radix Sort Algorithm

Radix Sort is a non-comparative sorting algorithm that sorts numbers digit by digit from least significant to most significant.

Time Complexity

Case Time Complexity
Best Case O(nk)
Average Case O(nk)
Worst Case O(nk)

Space Complexity

O(n + k) (due to auxiliary arrays for counting sort at each digit level)
