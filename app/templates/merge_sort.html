Merge Sort Algorithm

Merge Sort is a divide-and-conquer sorting algorithm that recursively splits the array into two halves, sorts them, and merges them back together.

Time Complexity

Case Time Complexity
Best Case (Already Sorted) O(n log n)
Average Case O(n log n)
Worst Case (Reverse Sorted) O(n log n)

Space Complexity

O(n) (for auxiliary space used during merging)
